# 阿尔兹海默症诊断系统 - 用户手册

## 1. 系统介绍

### 1.1 系统概述

**基于深度学习的多模态阿尔兹海默症分类与进展预测系统**

本系统是一个现代化的Web界面应用，采用FastAPI框架构建，旨在提供阿尔兹海默症的早期诊断和进展预测功能。系统通过融合多模态数据进行综合分析，为医生和研究人员提供辅助诊断工具。

### 1.2 系统特点

- 🧠 **多模态数据融合**：支持MRI影像、临床数据、分子生物标志物和生活方式数据的综合分析
- 📊 **深度学习模型**：基于PyTorch的多模态神经网络，提供准确的诊断结果
- 📋 **自动报告生成**：生成详细的PDF诊断报告，包含所有分析结果和可视化图表
- 🌐 **现代化Web界面**：响应式设计，支持桌面和移动设备
- 📱 **用户友好**：直观的操作界面，易于使用
- 💾 **数据管理**：支持历史报告管理和数据统计

### 1.3 适用人群

- **医生**：辅助诊断和制定治疗方案
- **研究人员**：分析研究数据和验证研究假设
- **医疗专业人员**：了解患者病情和进展风险
- **患者家属**：了解患者状况和护理建议

## 3. 诊断分析流程

### 3.1 使用示例数据

系统提供四种诊断类别的演示数据，方便用户熟悉系统功能：

1. **选择示例数据类型**：
   - 🟢 **CN (认知正常)**：正常生物标志物和神经心理学测试
   - 🔵 **EMCI (早期轻度认知障碍)**：边缘异常的生物标志物
   - 🟡 **LMCI (晚期轻度认知障碍)**：明显异常的生物标志物
   - 🔴 **AD (阿尔兹海默症)**：严重异常的生物标志物

2. **点击对应的按钮**：系统自动加载示例数据并进行分析

3. **查看分析结果**：系统会显示详细的诊断结果和可视化图表

### 3.2 上传真实数据

1. **准备数据**：
   - MRI图像：支持.nii、.nii.gz、.jpg、.png格式
   - 患者信息：年龄、性别、教育年限等
   - 生活方式数据：运动频率、睡眠时长、饮食习惯等
   - 病史信息：临床病史、家族史

2. **上传MRI图像**：
   - 点击"浏览文件"按钮选择MRI文件
   - 等待文件上传完成

3. **填写患者信息**：
   - 基本信息：年龄、性别、教育年限
   - 生活方式：运动频率、睡眠时长、饮食习惯等
   - 病史信息：临床病史、家族史

4. **开始分析**：
   - 点击"开始分析"按钮
   - 等待分析完成（可能需要几分钟）

5. **查看分析结果**：
   - 系统会显示详细的诊断结果和可视化图表

## 4. 诊断结果解读

### 4.1 诊断摘要

- **诊断类别**：CN（认知正常）、EMCI（早期轻度认知障碍）、LMCI（晚期轻度认知障碍）、AD（阿尔兹海默症）
- **诊断置信度**：系统对诊断结果的信心程度（0-100%）
- **12个月进展风险**：患者在未来12个月内病情进展的风险
- **综合风险等级**：低、中、高三级风险评估

### 4.2 脑部图像分析

- **脑部结构分析图**：显示脑部MRI图像，标注病变位置
- **脑区标注**：额叶、颞叶、顶叶、枕叶等关键脑区的标注和分析
- **病变位置**：显示可能的病变区域和严重程度

### 4.3 脑区风险热力图

- **热力图**：以颜色编码显示不同脑区的风险程度
- **颜色含义**：黄色→橙色→红色表示风险从低到高
- **风险评估**：针对每个脑区的风险评估和说明

### 4.4 12个月进展预测

- **风险趋势图**：显示未来12个月的风险变化趋势
- **月度风险数据**：详细的月度风险数据表
- **趋势分析**：对风险变化趋势的分析和说明

### 4.5 关键风险指标

- **风险指标列表**：显示重要的风险指标
- **检测值**：实际检测到的值
- **风险等级**：低、中、高三级风险评估
- **参考范围**：正常参考范围和说明

### 4.6 个性化指导建议

- **运动锻炼建议**：适合患者的运动方式和频率
- **饮食调理建议**：推荐的饮食方案和营养补充
- **认知训练建议**：认知功能训练方法
- **社交活动建议**：社交互动和心理支持
- **定期复查建议**：复查时间和项目建议

## 5. PDF报告生成

### 5.1 生成报告

1. **诊断完成后**：点击"生成PDF报告"按钮
2. **选择报告模板**：标准/详细/简洁
3. **设置报告参数**：
   - 语言：中文/英文
   - 字体：黑体/微软雅黑/Arial
   - 字号：小/中/大
   - 内容选项：患者信息、风险分析、建议、可视化图表
4. **点击"生成"**：系统开始生成PDF报告
5. **预览和下载**：生成完成后，系统会自动预览PDF报告，您可以下载或打印报告

### 5.2 报告内容

PDF报告包含以下内容：

- **封面页**：系统名称、患者信息、报告日期
- **患者基本信息**：姓名、年龄、性别、教育年限、联系方式、病史
- **诊断结果概要**：诊断结论、置信度、风险评分、风险等级
- **脑部图像分析**：脑结构分析图、额叶/颞叶/顶叶/枕叶说明
- **脑区风险热力图**：脑区热力图、颜色含义说明
- **关键风险指标**：5大风险指标表格（含检测值、风险等级、参考说明）
- **12个月进展预测**：风险趋势折线图、月度数据表、趋势分析
- **个性化指导建议**：按类别分类展示（运动、饮食、认知、社交、复查）
- **医生信息和报告说明**：医生信息、5条重要说明、报告编号

## 9. 常见问题与解决方案

### 9.1 登录问题

| 问题 | 解决方案 |
|------|---------|
| 忘记密码 | 联系系统管理员重置密码 |
| 登录失败 | 检查用户名和密码是否正确 |
| 账户被锁定 | 联系系统管理员解锁账户 |

### 9.2 上传问题

| 问题 | 解决方案 |
|------|---------|
| 文件上传失败 | 检查文件格式是否支持，文件大小是否超限 |
| 上传速度慢 | 检查网络连接，尝试使用较小的文件 |
| 上传后无法分析 | 检查文件是否损坏，尝试重新上传 |

### 9.3 分析问题

| 问题 | 解决方案 |
|------|---------|
| 分析时间过长 | 耐心等待，复杂分析可能需要较长时间 |
| 分析失败 | 检查输入数据是否完整，系统是否有足够的资源 |
| 分析结果异常 | 检查输入数据是否正确，尝试重新分析 |

### 9.4 报告问题

| 问题 | 解决方案 |
|------|---------|
| 报告生成失败 | 检查系统资源，确保有足够的存储空间 |
| 报告内容不完整 | 检查分析结果是否完整，尝试重新生成报告 |
| 报告下载失败 | 检查网络连接，尝试重新下载 |
